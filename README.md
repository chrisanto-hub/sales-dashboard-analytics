# Sales Dashboard Analytics

## Business Problem

Small businesses often lack visibility into their sales performance.
This project analyzes sales data to uncover revenue trends,
top-performing products, and category insights.

## Dataset

Sales dataset sourced from Kaggle (public dataset).

## Analysis Workflow

- Data cleaning and preprocessing (Python, Pandas)
- Feature engineering for time-based analysis
- KPI calculation
- Visual analytics

## Key Insights

- A small subset of products contributes a large share of total revenue.
- Sales show clear time-based trends useful for forecasting.
- Certain categories underperform and may require strategic review.

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Business Value

This analysis demonstrates how NexGEN Analytics supports
data-driven decision-making for business growth.
